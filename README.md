# DravyaOS — Free, offline-first pharmacy management software for Windows & Android

**Download the installer, updates, and Android APK for [DravyaOS](https://dravyaos.com).**
This repository is the public download bucket only — installers and the auto-updater
manifest. There is no source code here.

DravyaOS is a **free**, **offline-first**, keyboard-driven **pharmacy management app** for
Indian medical stores and chemists. Run the entire counter without the internet: **GST
billing**, **batch & expiry tracking**, customer credit (*udhaar / khata*), supplier
payments, and reports — all working offline, syncing automatically when a connection is
available.

## ⬇️ Download

| Platform | Link |

| **Windows** (10/11, 64-bit) | **[Download DravyaOS for Windows](https://github.com/dravyaos-admin/DravyaOS-dist/releases/latest/download/dravyaos-setup.exe)** |

automatically** from this repository — no reinstall needed.

> Windows may show a SmartScreen warning on first run because the installer isn't yet
> code-signed. Click **More info → Run anyway** to continue.

## Why DravyaOS

- **₹0, forever** — no subscription, no per-bill charge, no seat limits.
- **Works offline** — the full pharmacy runs on your PC with no internet; nothing stalls
  when the connection drops.
- **Keyboard-first billing** — clear the counter rush without touching the mouse.
- **GST-compliant invoices** — correct tax on every medical bill, ready to print.
- **Batch & expiry tracking** — see near-expiry stock at a glance and stop selling
  expired medicine.
- **Customer credit (udhaar / khata)** — track dues and payments per customer.
- **Suppliers & purchases** — record purchases, manage supplier payments and returns.
- **Reports** — sales, dues, expiry and stock reports whenever you need them.
- **Automatic sync** — data syncs safely across devices once you're back online.

See the full feature list and screenshots at **[dravyaos.com](https://dravyaos.com)**.

## Requirements

- **Windows 10 or 11** (64-bit), or **Android** 8.0+ for the tablet/phone build.
- No internet connection required to run — only for optional sync and updates.
